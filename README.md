# Handcraft Haven - E-Commerce Website

A beautiful, dark-themed e-commerce website for handcrafted products with WhatsApp ordering integration.

## 🛍️ Products

- **Handcrafted Notebook** - NPR 450
- **Wooden Pen Holder** - NPR 650  
- **Leather Wallet** - NPR 1,200
- **Handmade Candle** - NPR 350
- **Ceramic Mug** - NPR 550
- **Decorative Basket** - NPR 800

## ✨ Features

- 🌙 **Dark Theme** - Modern, elegant design
- 📱 **Mobile Responsive** - Works on all devices
- 💬 **WhatsApp Integration** - Direct ordering via WhatsApp
- 🎨 **Beautiful Animations** - Smooth transitions and effects
- 📖 **About Section** - Company story and mission

## 🚀 How to Run

1. Open the folder in your terminal
2. Run: `python -m http.server 8000`
3. Open your browser and go to: `http://localhost:8000`

## 📱 WhatsApp Ordering

Each product has a "Order via WhatsApp" button that:
- Opens WhatsApp with pre-filled message
- Includes product name and price
- Ready for customer to send order
